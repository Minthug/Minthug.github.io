# <span style="color:lightblue">컴퓨터란?</span>
> 입력 받은 데이터를 정해진 규칙을 통해서 데이터를 처리하고 저장 한 뒤 
출력하는 전자 장치
* ‘입력(input)’
컴퓨터는 사람으로부터 데이터를 받아들입니다.
<br>
* ‘정해진 규칙대로’
정해진 규칙은 프로그램을 의미
프로그램은 Software 및 Application 이라 지칭
<br>
* ‘처리해서’
정해진 프로그램 대로 연산을 수행합니다.
컴퓨터는 중앙처리장치(CPU)를 통해 연산을 수행.
<br>
* ‘처리 ➡️ 결과 ➡️ 저장’
컴퓨터는 데이터를 저장할 수 있는 장치 보유 하고 있다.
처리 결과물을 기억장치(RAM,SSD)등에 저장합니다.
<br>
* ‘출력(output)’
컴퓨터는 데이터 처리 결과를 사람이 이해 할수있는 형태의 정보로 
사람에게 보여준다.

## 💡 캐시 메모리 (Cache Memory)
> 캐시 메모리는 CPU의 처리속도와 주기억장치의 접근 속도 차이를 줄이기 위해 사용하는 고속 Buffer Memory(✶완충기억장치) 입니다.
<br>캐시 메모리는 한 번 CPU가 사용한 데이터와 관련된 데이터 중, 재사용 가능성이 높은 데이터를 주기억장치 로부터 미리 불러와서 대기 시켜 놓는 역할을 합니다.

주 기억장치를 ATM, 캐시를 지갑이라고 이해해주세요.
지갑에 있는 현금은 바로바로 꺼내서 사용 할 수 있으니까


# <span style="color:lightblue">프로그래밍이란?</span>
> 프로그램을 만드는 과정으로써, 특정 목적을 이루기 위해 설계한 알고리즘을 프로그래밍 언어로써 작성하는 과정을 칭한다.

```
b8 21 0a 00 00 a3 0c 10 00 06 b8 6f 72 6c 64 a3 08 10 00 06 b8 6f 2c 20 57
a3 04 10 00 06 b8 48 65 6c 6c a3 00 10 00 06 b9 00 10 00 06 ba 10 00 00 00
bb 01 00 00 00 b8 04 00 00 00 cd 80 b8 01 00 00 00 cd 80
```
위 글은 "Hello World"를 출력하는 기계어 이다.
사람이 이렇게 작성할수 없으니, 사람과 기계 간의 매개체가 필요하다.

그것이 바로 '프로그래밍 언어'이다.
```
System.out.println("Hello world");
```
자바를 통해 "Hello world"를 컴퓨터에게 출력 명령하는 코드이다.

<span style="color:powderblue">기계어와 같이 컴퓨터에게 가까운 언어일수록 저수준 언어라고 하며, 
반대로 Java 및 C언어와 같이 사람과 가까운 언어일수록 고수준 언어라고 한다.</span>

## 빌드(bulid)
>컴파일을 마친 코드는 기계어 코드로 번역만 되어서 컴퓨터가 이해할 수 있을 뿐, 아직 사용자가 소스 코드를 실행할 수는 없습니다. 
프로그램이 실행되려면 .exe또는 .msi 확장자를 가진 실행 파일로 만들어주는 빌드 과정이 필요_!_
빌드가 완료되야 소스코드를 실행시킬 수 있는 하나의 프로그램으로 완성됩니다.

### 통합개발환경 (Integrated Development Environment, IDE)

>IDE는 코드를 작성할 때 사용하는 프로그램으로써, 프로그래밍에 필요한 모든 작업을 하나의 프로그램 안에서 처리할 수 있도록 만들어진 프로그램입니다. 
또한 단순히 코드를 작성하고 편집하는 것뿐만 아니라, 디버깅, 빌드 등의 작업을 하나의 IDE 안에서 할 수 있습니다. 
* Visual Studio Code
* IntelliJ
* Eclipse
